## 運用ルール
- 各フォルダの**プロジェクト/**には、進行中のタスクを格納。
- **ドキュメント/**には、仕様書やマニュアルを保存。
- **ツール/**には、スクリプトやテンプレートを置く。
- 共有リソースは全社員が参照可能。変更時はチームに共有。
- スキル開発は**スキル・人材開発/**で管理し、定期的に更新。

## ローカルサーバー設定ガイド

### セットアップ手順（初回のみ）

#### Step 1: hosts ファイルを編集

1. メモ帳を**管理者として実行**
2. `C:\Windows\System32\drivers\etc\hosts` を開く
3. ファイルの最後に以下を追加：
   ```
   127.0.0.1 k3-works.local
   ```
4. 保存して閉じる

### サーバー起動方法

#### 方法1：バッチファイルで起動（推奨）

1. `start-server.bat` をダブルクリック
2. ブラウザで `http://k3-works.local:8000` にアクセス

#### 方法2：PowerShell で起動

```powershell
cd c:\Users\user\Documents\K3Works
python -m http.server 8000
```

その後、ブラウザで `http://k3-works.local:8000` にアクセス

### サーバー停止

コマンドプロンプト/PowerShell で `Ctrl+C` を押す
