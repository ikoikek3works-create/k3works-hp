# K3Works ローカルサーバー セットアップ完了チェックリスト

このファイルを参考に、セットアップを進めてください。

---

## ✅ Step 1: hosts ファイル設定（管理者権限が必要）

### やること
1. **メモ帳を管理者として実行**
   - スタートメニュー → 「メモ帳」を右クリック → 「管理者として実行」

2. **hosts ファイルを開く**
   - メニューの「ファイル」→「開く」
   - ファイル名指定：`C:\Windows\System32\drivers\etc\hosts`
   - ファイルの種類を「すべてのファイル」に変更

3. **該当行を追加**
   ファイルの最後に以下を追加：
   ```
   127.0.0.1 k3-works.local
   ```

4. **保存**
   - Ctrl+S で保存

### 確認方法
コマンドプロンプトで以下を実行：
```
ping k3-works.local
```
`127.0.0.1` から応答があれば成功

---

## ✅ Step 2: Python インストール確認

K3Works フォルダで PowerShell を開いて以下を実行：

```powershell
python --version
```

Python 3.x がインストールされていれば OK。

**インストール不要な場合**: Windows 11 なら既に Python が入っている可能性があります。

---

## ✅ Step 3: サーバー起動

### 方法1：バッチファイルで起動（推奨）

1. **start-server.bat** をダブルクリック
2. コマンドプロンプトが開いて以下が表示される：
   ```
   ====================================
   K3Works ローカルサーバー起動中...
   ====================================
   
   ブラウザで以下のURLにアクセス：
   http://k3-works.local:8000
   ```

3. ブラウザで `http://k3-works.local:8000` にアクセス

### 方法2：PowerShell で手動起動

K3Works フォルダで PowerShell を開いて：

```powershell
python -m http.server 8000
```

コンソールに以下が表示される：
```
Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/) ...
```

---

## ✅ Step 4: ブラウザでプレビュー

上記のサーバー起動後、ブラウザで以下にアクセス：

```
http://k3-works.local:8000
```

美しく整形された K3Works のホームページが表示されます！

## トラブルシューティング

### ❌ エラー：「k3-works.local にアクセスできません」
→ hosts ファイルに `127.0.0.1 k3-works.local` が追加されているか確認

### ❌ エラー：「 Python が見つかりません」
→ Python がインストールされていません。Microsoft Store または python.org からインストール

### ❌ エラー：「ポート 8000 は既に使用中」
→ 別のアプリが使用中。別のポートで起動：
```powershell
python -m http.server 9000
```

## サーバー停止方法

コマンドプロンプト/PowerShell で：
```
Ctrl+C
```
